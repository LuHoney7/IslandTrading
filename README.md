# 岛买岛卖（LslandTrading）

**岛买岛卖**是一款面向全国范围内高校学生的网上二手交易平台。
 
- **功能丰富** ：支持广告发布，发布商品，购买商品；
- **特色界面** ：地图画的商品浏览界面，创新，独特；
